# mikrotik-diagrams
mikrotik diagrams for draw.io

## How Draw.io Diagrams Work

### Summary Page

Once draw.io is open, you can import pictograms (XML files) by drag and drop them, or go to "Notepad" on the left ➡️ Edit (pencil icon) ➡️ Import (XML file "PictoMikrotik.xml").

### Link Pictograms

The pictograms are named and have connection points on their eth port. If you need to add more, right-click on the element and select "Edit Connection Points".
